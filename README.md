# 古着物ドレス Makuake掲載シミュレーション

古着物を一点もののドレスへ仕立て直すプロジェクトの、Makuake掲載ページを想定した静的プロトタイプです。実際のMakuakeや決済システムとは接続していません。

## 起動

PowerShellの実行ポリシーに影響されにくい `serve.bat` がおすすめです。

```powershell
.\serve.bat
```

PowerShellスクリプトで起動する場合は次を実行します（Codex同梱Pythonにも自動対応します）。

```powershell
.\serve.ps1
```

ブラウザで `http://127.0.0.1:4173/` を開いてください。
`serve.ps1` が実行ポリシーで止まる場合は、`serve.bat` または VS Code の Live Server で `index.html` を開いてください。

## 実装済み

- Makuake型の商品詳細レイアウト（支援状況、タブ、長尺ストーリー、リターン列）
- PC／スマートフォン対応
- 「気になる」、シェアURLコピー、リターン選択のシミュレーション
- スクロール位置に応じたタブ表示
- AI生成による古着物ドレスの仮ビジュアル

## ファイル構成

```text
assets/       商品・制作工程の仮画像
index.html    ページ本体
styles.css   レスポンシブデザイン
script.js    シミュレーション操作
serve.ps1    ローカルプレビュー起動
serve.bat    実行ポリシー回避用のローカルプレビュー起動
```

## バージョン管理の方針

- `main`: 閲覧可能な安定版
- 機能追加は `feature/<内容>` ブランチで作業
- コミットは「構成」「ビジュアル」「文言」などレビュー可能な単位に分ける
- 実写真や確定コピーへの差し替えも履歴が追えるよう、上書き前にコミットする

## 画像生成プロンプト概要

- ヒーロー: 古着物の藍・朱・金糸を生かした一点ものドレス、京都の静かなアトリエ、自然光のエディトリアル写真
- 制作工程: 古着物と帯地を手で縫い合わせる工程、素材感と手仕事が伝わる接写
