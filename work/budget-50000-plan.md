# 5万円枠で進めるMakuake準備プラン

古着物ドレスそのものの制作・古着物調達・完成までは制作者側で完結する前提。ここでは、Makuake出品に向けて「自分でやる／友人に協力してもらう」ことで、5万円以内に収める現実的な使い方を整理する。

## 結論

5万円では、外注でフル制作するのではなく「本番前の検証」と「最低限の見せ場づくり」に集中する。

優先順位は次の通り。

1. 実物写真・着用写真を最低限撮る
2. 制作者インタビューをページ本文に反映する
3. SNSで事前反応を見る
4. Makuake本番で差し替えられる素材管理を整える

## 予算配分案

| 項目 | 目安 | 内容 |
|---|---:|---|
| モデル・協力者への謝礼 | 10,000〜15,000円 | 友人2〜3名に交通費＋お礼。年代・性別の幅を出す |
| 撮影場所・移動・小物 | 8,000〜12,000円 | カフェ、ギャラリー風スペース、花、小物、交通費 |
| ヘアメイク・着付け補助 | 0〜10,000円 | 自前＋友人協力。必要ならポイントメイクだけ依頼 |
| 広告テスト | 10,000〜15,000円 | Instagram/Facebookで写真A/Bテスト |
| 印刷・備品・予備費 | 5,000〜8,000円 | 同意書印刷、クリップ、スチーマー、簡易照明など |

合計目安：33,000〜50,000円

## 自分でやること

- 制作者インタビューの質問作成
- インタビュー録音・文字起こし
- Makuakeページ本文の下書き
- 撮影構図リスト作成
- 友人モデルへの依頼文作成
- Google Drive等で写真・同意書を整理
- SNS投稿文、事前告知文の作成

## 友人にお願いできること

- 年代別・性別別の着用モデル
- スマホまたはカメラでの撮影補助
- ヘアメイク、アクセサリー、小物協力
- 撮影場所候補の提供
- ページを見た率直な感想

## 最低限撮るべき写真

1. メイン着用写真：全身、縦構図
2. 横長ヒーロー写真：Makuakeキービジュアル想定
3. 制作者の手元：ほどく、裁つ、縫う
4. 生地アップ：柄、織り、金糸、経年の質感
5. 年代違いの着用：20代、30〜40代、50代以上のどれか2〜3名
6. 性別を限定しない着こなし：羽織・ジャケット・パンツ合わせ

## 5万円枠でやらないこと

- プロカメラマンの終日拘束
- スタジオ撮影一式
- 本格動画制作
- 大規模広告
- 外注ライティング

## Codexで支援すること

- インタビュー質問リスト作成
- 制作者の語りをMakuake向け本文へ編集
- 撮影指示書作成
- モデル別シーン案作成
- SNS投稿案作成
- 公開サイト更新
- 画像差し替え・GitHub Pages反映

## 次のアクション

1. 制作者インタビューを1時間実施
2. 友人モデル候補を3名決める
3. 撮影日を半日だけ確保
4. 現在の仮画像を見ながら、撮りたい構図を10カットに絞る
5. 実写素材が揃ったらサイト画像を差し替える
